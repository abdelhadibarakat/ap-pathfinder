# Academic Pathway System

## Overview
This project is a Python-based academic planning system that helps students create personalized AP course pathways based on their academic profile.

The program considers:
- GPA or SAT scores
- Previously completed AP exams
- Engineering major of interest
- AP prerequisite relationships

It then recommends an optimized AP sequence and can estimate future AP performance.

## Features
- Input validation
- Personalized AP recommendations
- Course prerequisite logic
- SAT/GPA-based recommendations
- AP score prediction
- Modular function-based design

## How to Run
1. Ensure you have [Python](https://www.python.org/downloads/) installed.
2. Clone this repository:
   `git clone https://github.com/AbdelhadiBarakat/AP-Pathfinder.git`
3. Navigate to the folder:
   `cd AP-Pathfinder`
4. Run the script:
   `python AP-Pathfinder.py`
